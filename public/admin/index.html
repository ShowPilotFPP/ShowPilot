Interaction Safeguards
Jukebox Queue Depth
Maximum pending requests in the queue. Use 0 for unlimited.
Jukebox Sequence Request Limit
Max times the same sequence can be in the queue. Use 0 to allow unlimited.
Prevent Multiple Requests
Blocks a viewer from making more than one request while their song is queued or playing.
Prevent Multiple Votes
Blocks a viewer from voting more than once per round.
Reset Votes After Round
Resets all vote counts to zero after each voting round. If off, votes carry forward.
Play PSA
Plays a PSA sequence after a specified number of viewer interactions.
PSA Frequency
Play a PSA every N viewer interactions (votes or requests). Mark sequences as PSA via the Sequences list.
Check Viewer Present
Requires viewers to be near the show (via GPS) before they can vote or request. Strict mode — blocks if check fails.
Show Location
GPS coordinates of the show. Use the "Detect" button from a device at the show location, or paste in manually.
Check Radius (miles)
How close viewers must be to the show location. Default 0.5 miles.
Hide Sequence After Played
After a requested sequence finishes, hide it from the list until this many other sequences have played. Use 0 to never hide.
Block Viewer IP Addresses
Comma-separated list of IPs that can't vote or request. Use this to block abusers.
